\section*{Finalidad del Tratamiento de Datos}

La aplicación móvil \textbf{Planty} es un proyecto integrador con fines exclusivamente \textbf{académicos y educativos}, desarrollado por estudiantes de la Universidad Politécnica de Chiapas. Los datos personales que se recaben serán utilizados para las siguientes finalidades:

\subsection*{Finalidades Primarias (Necesarias):}
\begin{itemize}[leftmargin=*]
    \item Crear y administrar la cuenta de usuario en la aplicación
    \item Proporcionar acceso a las funcionalidades de gestión de huertos urbanos
    \item Generar recomendaciones personalizadas mediante algoritmos genéticos
    \item Enviar notificaciones relacionadas con el cuidado de plantas y huertos
    \item Brindar asistencia mediante el chatbot inteligente (IA local)
    \item Almacenar y gestionar la información de los huertos del usuario
\end{itemize}

\subsection*{Finalidades Secundarias (No necesarias):}
\begin{itemize}[leftmargin=*]
    \item Mejorar la experiencia del usuario mediante análisis de uso
    \item Desarrollo y evaluación del proyecto integrador
\end{itemize}

Si no desea que sus datos personales sean tratados para las finalidades secundarias, puede manifestarlo enviando un correo a: bs.personal.0001@gmail.com

\section*{Datos Personales que se Recaban}

\subsection*{Datos de Identificación y Contacto:}
\begin{itemize}[leftmargin=*]
    \item \textbf{Nombre completo}
    \item \textbf{Correo electrónico}
    \item \textbf{Contraseña} (almacenada de forma cifrada con algoritmos seguros)
    \item \textbf{Imagen de perfil} (opcional)
    \item \textbf{Fecha de registro}
\end{itemize}

\subsection*{Datos de Uso de la Aplicación:}
\begin{itemize}[leftmargin=*]
    \item Nivel de experiencia en cultivo de plantas
    \item Información de huertos creados (nombre, dimensiones, descripción)
    \item Información de plantas (nombre, especie, tipo, requerimientos)
    \item Historial de actividades en la aplicación
    \item Preferencias de configuración
    \item Token de dispositivo para notificaciones push (Firebase Cloud Messaging)
\end{itemize}

\subsection*{Datos Técnicos:}
\begin{itemize}[leftmargin=*]
    \item Identificador único de usuario (UUID)
    \item Token de sesión (para autenticación)
    \item Fecha y hora de uso de la aplicación
\end{itemize}

\textbf{Importante:} \textcolor{red}{NO se recopilan datos de ubicación GPS, datos biométricos, ni información financiera.}

\section*{Datos Sensibles}

De conformidad con la Ley Federal de Protección de Datos Personales en Posesión de los Particulares, se consideran datos sensibles únicamente:

\begin{itemize}[leftmargin=*]
    \item \textbf{Contraseña de acceso} (almacenada con cifrado de alta seguridad)
\end{itemize}

Los demás datos recopilados \textbf{NO se consideran sensibles} conforme a la ley, ya que no revelan origen étnico, estado de salud, información genética, creencias religiosas, filosóficas o morales, afiliación sindical, opiniones políticas o preferencia sexual.

\section*{Transferencia de Datos}

Sus datos personales pueden ser compartidos con los siguientes terceros, únicamente para las finalidades descritas:

\begin{enumerate}[leftmargin=*]
    \item \textbf{Firebase Cloud Messaging (Google):} Para el envío de notificaciones push sobre recordatorios de riego, alertas climáticas y recomendaciones de huertos.

    \item \textbf{Servicios de almacenamiento en nube:} Para el resguardo seguro de la información de la aplicación.
\end{enumerate}

\textbf{Importante:}
\begin{itemize}[leftmargin=*]
    \item El chatbot funciona con IA local (Llama 3.2), por lo que las conversaciones \textbf{NO se envían a servicios externos} como OpenAI o Google.
    \item Las conversaciones del chat \textbf{NO se almacenan} en el servidor, solo temporalmente en el dispositivo del usuario.
\end{itemize}

\section*{Medidas de Seguridad}

Para proteger sus datos personales, implementamos:

\begin{itemize}[leftmargin=*]
    \item Cifrado de contraseñas mediante algoritmos seguros (bcrypt/hash)
    \item Comunicación segura mediante HTTPS
    \item Tokens de autenticación JWT (JSON Web Tokens)
    \item Almacenamiento seguro en dispositivo (Secure Storage)
    \item Arquitectura de microservicios con autenticación
    \item Validación de entrada de datos para prevenir inyección SQL y XSS
\end{itemize}

\section*{Período de Conservación}

Los datos personales serán conservados únicamente \textbf{durante el período del proyecto integrador} (enero 2025 - junio 2025). Posterior a esta fecha, los datos serán eliminados de forma segura, salvo que el usuario solicite la eliminación antes.
