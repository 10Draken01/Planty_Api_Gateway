    \fontsize{11}{14}\selectfont
    Los sistemas de gestión agrícola tradicionales presentan limitaciones significativas:
    falta de integración entre diferentes módulos, ausencia de asistencia inteligente
    personalizada, gestión ineficiente de huertos y plantas, y optimización manual de
    recursos. Para resolver estos problemas, se desarrollará una plataforma integral
    basada en microservicios que incluirá gestión completa de usuarios y autenticación
    segura, un chatbot inteligente con RAG (Retrieval-Augmented Generation) que aprende
    de documentos, administración de huertos con seguimiento en tiempo real, y un algoritmo
    genético para optimizar la configuración de plantas considerando compatibilidad entre
    especies, eficiencia hídrica y utilización de espacio. Esta solución mejorará la
    productividad agrícola, reducirá errores de planificación, optimizará el uso de recursos,
    y proporcionará asistencia personalizada a agricultores. El desarrollo se completará en
    un plazo de dos meses.
\end{minipage}

\begin{minipage}[t]{0.55\textwidth}
    \colorbox{darkblue}{
        \parbox{0.95\textwidth}{
            \color{white}\fontsize{14}{17}\selectfont\bfseries Descripción
        }
    }

    \vspace{0.3cm}

    Plataforma de microservicios para gestión agrícola inteligente. Permite a los agricultores
    gestionar usuarios, autenticarse de forma segura, administrar huertos completos, interactuar
    con un chatbot con IA que aprende de documentos PDF, y optimizar configuraciones de plantas
    usando algoritmos genéticos. Todo orquestado a través de un API Gateway centralizado.

    \vspace{0.5cm}

    \colorbox{darkblue}{
        \parbox{0.95\textwidth}{
            \color{white}\fontsize{14}{17}\selectfont\bfseries Características
        }
    }

    \vspace{0.3cm}

    \begin{itemize}[leftmargin=*,itemsep=0pt]
        \item \textbf{Plataforma}: Web API RESTful + Microservicios Docker
        \item \textbf{Microservicios principales}:
        \begin{itemize}[itemsep=0pt]
            \item \textbf{API Gateway}: Enrutador centralizado con seguridad
            \item \textbf{API Users}: Gestión completa de usuarios
            \item \textbf{Authentication}: Autenticación JWT
            \item \textbf{API Chatbot}: Chat inteligente con RAG + Ollama
            \item \textbf{API Orchard}: Gestión de huertos y plantas
            \item \textbf{PlantGen}: Algoritmos genéticos multi-objetivo
        \end{itemize}
        \item \textbf{Base de datos}: MongoDB + ChromaDB (vectorial)
        \item \textbf{IA/ML}: Ollama (llama3.2 + nomic-embed-text)
        \item \textbf{Arquitectura}: Clean Architecture + DDD
        \item \textbf{Seguridad}: JWT, bcryptjs, rate limiting, CORS, Helmet
        \item \textbf{Orquestación}: Docker Compose
    \end{itemize}

    \vspace{0.3cm}

    \textbf{Tecnologías utilizadas}: Node.js + Express (Backend primario),
    Python + FastAPI (Algoritmos genéticos), TypeScript, MongoDB, ChromaDB,
    Ollama (LLM local), Docker.
