\begin{minipage}[t]{0.7\textwidth}
    \vspace{1cm}
    \large
    Los sistemas de gestión agrícola tradicionales presentan limitaciones significativas:
    falta de integración entre diferentes módulos, ausencia de asistencia inteligente
    personalizada, interfaces móviles deficientes, gestión ineficiente de huertos y plantas,
    y optimización manual de recursos. Para resolver estos problemas, se desarrollará una
    \textbf{plataforma integral full-stack} que incluye: \textbf{Backend} con 6 microservicios
    (API Gateway, Authentication, Users, Chatbot con RAG, Orchards, Algoritmos Genéticos),
    y \textbf{Frontend} multiplataforma en Flutter (Android, iOS, Web, Desktop) con arquitectura
    limpia, autenticación JWT, chat inteligente con IA local (Ollama), gestión de huertos,
    y sistema de diseño neumórfico. Esta solución mejorará la productividad agrícola, reducirá
    errores de planificación, optimizará el uso de recursos, y proporcionará asistencia
    personalizada a agricultores desde cualquier dispositivo. El desarrollo se completará en
    un plazo de tres meses.
\end{minipage}

\begin{center}

\vspace{1cm}

\begin{minipage}[t]{0.55\textwidth}
    \colorbox{darkblue}{
        \parbox{0.95\textwidth}{
            \color{white}\Large\bfseries Descripción
        }
    }

    \vspace{0.3cm}

    Plataforma full-stack de microservicios para gestión agrícola inteligente. \textbf{Backend}
    con arquitectura de microservicios orquestados por Docker, APIs REST, bases de datos
    MongoDB + ChromaDB, e IA local con Ollama. \textbf{Frontend} multiplataforma en Flutter
    con Clean Architecture, MVVM, Atomic Design, autenticación JWT persistente, y diseño
    Material 3 adaptativo. Sistema completo integrado para agricultores profesionales.

    \vspace{0.5cm}

    \colorbox{darkblue}{
        \parbox{0.95\textwidth}{
            \color{white}\Large\bfseries Características
        }
    }

    \vspace{0.3cm}

    \textbf{BACKEND (Node.js/Python):}
    \begin{itemize}[leftmargin=*,itemsep=0pt]
        \item 6 microservicios independientes
        \item API Gateway centralizado
        \item Autenticación JWT
        \item MongoDB + ChromaDB (vectorial)
        \item IA local (Ollama + RAG)
        \item Algoritmos genéticos multi-objetivo
        \item Docker Compose orquestado
    \end{itemize}

    \textbf{FRONTEND (Flutter):}
    \begin{itemize}[leftmargin=*,itemsep=0pt]
        \item Multiplataforma (Android, iOS, Web, Desktop)
        \item Clean Architecture + MVVM
        \item Provider (State Management)
        \item GoRouter con rutas protegidas
        \item Atomic Design System
        \item Material 3 + Neumorphic UI
        \item SecureStorage para tokens
        \item 59 archivos Dart organizados
    \end{itemize}

    \textbf{Tecnologías}: Node.js, Express, Python, FastAPI, Flutter, Dart,
    TypeScript, MongoDB, ChromaDB, Ollama, Docker, JWT.
